# Atoms

Atoms are the foundational building blocks of a user interface. They include basic HTML elements such as buttons, inputs and icons.

## Button

Buttons allow users to trigger an action or event with a single click.

```javascript
import { Button } from '@terenceodonoghue/react-components/velocity';
```

The `variant` prop defines one of three styles: primary (default), outline, or minimal. Standard [button](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button) attributes are also supported, e.g. `disabled`.

<Canvas>
  <Story name="Button">
    <Button variant="primary">Primary</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="minimal">Minimal</Button>
    <Button variant="primary" disabled>
      Primary
    </Button>
    <Button variant="outline" disabled>
      Outline
    </Button>
    <Button variant="minimal" disabled>
      Minimal
    </Button>
  </Story>
</Canvas>

## Radio

Radio buttons allow the user to select one option from a set.

```javascript
import { Radio } from '@terenceodonoghue/react-components/velocity';
```

Text can be placed to the right of the input using the `label` prop, alongside any standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes, e.g. `name`, `defaultChecked` or `disabled`.

<Canvas>
  <Story name="Radio">
    <Radio label="Option A" name="radio-group-1" defaultChecked />
    <Radio label="Option B" name="radio-group-1" />
    <Radio label="Option C" name="radio-group-2" defaultChecked disabled />
    <Radio label="Option D" name="radio-group-2" disabled />
  </Story>
</Canvas>

## Slider

Standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes are supported, e.g. `min`, `max`, `step` or `disabled`.

## Avatar

An avatar component represents an object or entity, typically a user's profile picture.

```javascript
import { Avatar } from '@terenceodonoghue/react-components/velocity';
```

The `variant` prop defines one of two styles: rounded (default) or squared. Standard [img](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img) attributes are also supported, e.g. `alt`.

<Canvas>
  <Story name="Avatar">
    <Avatar alt="hello" src={faker.image.avatar()} />
    <Avatar src={faker.image.avatar()} variant="squared" />
  </Story>
</Canvas>

Height and width can be customised by providing a number to the `size` prop.

<Canvas>
  <Story name="AvatarSize">
    <Avatar size={32} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={48} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={64} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={80} src={faker.image.avatar()} variant="rounded" />
  </Story>
</Canvas>

## Indicator

An indicator appends a simple coloured dot to the top-right of its child(ren).

```javascript
import { Indicator } from '@terenceodonoghue/react-components/velocity';
```

By default the indicator uses the primary theme colour and is always visible, but these can be controlled using the `color` and `visible` props.

<Canvas>
  <Story name="Indicator">
    <Indicator>
      <Avatar size={48} src={faker.image.avatar()} variant="rounded" />
    </Indicator>
    <Indicator color="#33AC2E">
      <Avatar size={48} src={faker.image.avatar()} variant="rounded" />
    </Indicator>
    <Indicator>
      <Notifications color="#BFC5D2" size={32} />
    </Indicator>
    <Indicator hidden>

    </Indicator>
  </Story>
</Canvas>

## Pill

Pills are simple labels with a background color.

```javascript
import { Pill } from '@terenceodonoghue/react-components/velocity';
```

Set the text value using the `label` prop, alongside `color` to set both the text and backdrop colours. The backdrop has a default opacity of `0.2`, but this can be adjusted with the `opacity` prop.

<Canvas>
  <Story name="Pill">
    <Pill color="#2E5BFF" label={76} />
    <Pill color="#8C54FF" label={69} />
    <Pill color="#00C1D4" label={28} />
    <Pill color="#F7C137" label={198} />
  </Story>
</Canvas>

## Card

Cards contain content and actions about a single subject.

```javascript
import { Card } from '@terenceodonoghue/react-components/velocity';
```

An optional `heading` prop renders text above the content block.

<Canvas data-has-bg>
  <Story name="Card">
    <Card heading="Heading">Content</Card>

  </Story>

## Box

A box is a simple wrapper for separating content.

```javascript
import { Box } from '@terenceodonoghue/react-components/velocity';
```

The default padding is `20`, but this can be adjusted with `p`, `py` or `px`.

<Canvas>
  <Story name="Box">
    <Box>Beep</Box>
    <Box p={48}>Boop</Box>
    <Box py={48}>Beep</Box>
    <Box px={48}>Boop</Box>
  </Story>
</Canvas>

## Icons

Icons can be used to improve visual interest and draw attention to an element.

```javascript
import { Icon } from '@terenceodonoghue/react-components/velocity';
import /* { IconName } */  from '@terenceodonoghue/react-icons/velocity';
```

Set the icon using `with`, and provide an optional `color`. This defaults to `currentColor` if left blank.

<Canvas>
  <Story name="Icons">
    <Icon with={Dashboard} />
    <Icon color="#2E5BFF" with={Analytics} />
    <Icon color="#33AC2E" with={Vehicles} />
    <Icon color="#8C54FF" with={Service} />
    <Icon color="#D63649" with={Map} />
    <Icon color="#00C1D4" with={Mail} />
    <Icon color="#F7C137" with={Settings} />
  </Story>
</Canvas>

A `backdrop` can be used for added emphasis. This defaults to `0.15` if left blank.

<Canvas>
  <Story name="IconsBackdrop">
    <Icon backdrop color="#33AC2E" with={Check} />
    <Icon backdrop color="#00C1D4" with={Marker} />
    <Icon backdrop color="#8C54FF" with={Energy} />
    <Icon backdrop color="#F7C137" with={Dynamic} />
  </Story>
</Canvas>

Icon size is `24` by default (or `20` with a backdrop) but can be adjusted with the `size` prop.

<Canvas>
  <Story name="IconsSize">
    <Icon with={Vehicles} />
    <Icon with={Vehicles} size={32} />
    <Icon with={Vehicles} size={40} />
    <Icon with={Vehicles} size={48} />
  </Story>
</Canvas>

There's also a `compact` size preset that reduces the default icon size and (if present) its backdrop.

<Canvas>
  <Story name="IconsCompact">
    <Icon backdrop color="#33AC2E" size="compact" with={Check} />
    <Icon backdrop color="#00C1D4" size="compact" with={Marker} />

  </Story>

## List

Lists are continuous, vertical indexes of labelled text.

```javascript
import { List } from '@terenceodonoghue/react-components/velocity';
```

A collection of `items`, each with a `label` and a `value`, makes up a list. Standard [dl](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dl) attributes are also supported.

<Canvas>
  <Story name="List">
    <List
      items={[
        { label: 'email', value: faker.internet.exampleEmail() },
        { label: 'phone', value: faker.phone.number('+144–####–####') },
        {
          label: 'location',
          value: `${faker.address.cityName()}, ${faker.address.stateAbbr()}`,
        },
      ]}
    />
  </Story>
</Canvas>
