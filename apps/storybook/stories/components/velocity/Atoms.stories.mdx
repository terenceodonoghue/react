# Atoms

Atoms are the foundational building blocks of a user interface. They include basic HTML elements such as buttons, inputs and icons.

## Button

Buttons allow users to trigger an action or event with a single click.

```javascript
import { Button } from '@terenceodonoghue/react-components/velocity';
```

The `variant` prop defines one of three styles: `primary` (default), `outline`, or `minimal`. Standard [button](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button) attributes are also supported, e.g. `disabled`.

<Canvas>
  <Story name="Button">
    <Button variant="primary">Primary</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="minimal">Minimal</Button>
    <Button variant="primary" disabled>
      Primary
    </Button>
    <Button variant="outline" disabled>
      Outline
    </Button>
    <Button variant="minimal" disabled>
      Minimal
    </Button>
  </Story>
</Canvas>

## Checkbox

Checkboxes allow the user to select one or more items from a set.

```javascript
import { Checkbox } from '@terenceodonoghue/react-components/velocity';
```

Text or other elements can be placed inside of the content block. Standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes are supported, e.g. `name`, `defaultChecked` or `disabled`.

<Canvas>
  <Story name="Checkbox">
    <Checkbox defaultChecked>Option A</Checkbox>
    <Checkbox>Option B</Checkbox>
    <Checkbox defaultChecked disabled>
      Option C
    </Checkbox>
    <Checkbox disabled>Option D</Checkbox>
  </Story>
</Canvas>

## Radio

Radio buttons allow the user to select one option from a set.

```javascript
import { Radio } from '@terenceodonoghue/react-components/velocity';
```

Text can be placed to the right of the input using the `label` prop, alongside any standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes, e.g. `name`, `defaultChecked` or `disabled`.

<Canvas>
  <Story name="Radio">

    <Radio label="Option B" name="radio-group-1" />
    <Radio label="Option C" name="radio-group-2" defaultChecked disabled />
    <Radio label="Option D" name="radio-group-2" disabled />
  </Story>
</Canvas>

## Slider

Sliders allow users to make selections from a range of values.


Standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes are supported, e.g. `min`, `max`, `step` or `disabled`.

Standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes are supported, e.g. `defaultChecked` or `disabled`.

## TextField

Text fields let users enter and edit text.


Text can be placed above the input using the `label` prop, alongside any standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes, e.g. `placeholder`, `type` or `disabled`.

## Avatar

An avatar represents an object or entity, typically a user's profile picture.

```javascript
import { Avatar } from '@terenceodonoghue/react-components/velocity';
```

The `variant` prop defines one of two styles: `rounded` (default) or `squared`. Standard [img](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img) attributes are also supported, e.g. `alt`.

<Canvas>
  <Story name="Avatar">
    <Avatar src={faker.image.avatar()} variant="rounded" />
    <Avatar src={faker.image.avatar()} variant="squared" />
  </Story>
</Canvas>

Height and width can be customised with the `size` prop.

<Canvas>
  <Story name="AvatarSize">
    <Avatar size={32} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={48} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={64} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={80} src={faker.image.avatar()} variant="rounded" />
  </Story>
</Canvas>

## Indicator

An indicator appends a simple coloured dot to the top-right of its child(ren).

```javascript
import { Indicator } from '@terenceodonoghue/react-components/velocity';
```

By default the indicator uses the primary theme colour and is always visible, but these can be controlled using the `color` and `visible` props.

<Canvas>
  <Story name="Indicator">
    <Indicator>
      <Avatar size={48} src={faker.image.avatar()} variant="rounded" />
    </Indicator>
    <Indicator color="#33AC2E">
      <Avatar size={48} src={faker.image.avatar()} variant="rounded" />
    </Indicator>
    <Indicator>
      <Notifications color="#BFC5D2" size={32} />
    </Indicator>
    <Indicator visible={false}>

    </Indicator>
  </Story>
</Canvas>

## List

Lists are continuous, vertical indexes of labelled text.

```javascript
import { List } from '@terenceodonoghue/react-components/velocity';
```

A collection of `items`, each with a `label` and a `value`, makes up a list. Standard [dl](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dl) attributes are also supported.

<Canvas>

      ]}
