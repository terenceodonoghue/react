# Atoms

Atoms are the foundational building blocks of a user interface. They include basic HTML elements such as buttons, inputs and icons.

## Button

Buttons allow users to trigger an action or event with a single click.

```javascript
import { Button } from '@terenceodonoghue/react-components/velocity';
```

The `variant` prop defines one of three styles: contained (default), outlined, or text. Standard [button](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button) attributes are also supported, e.g. `disabled`.

<Canvas>
  <Story name="Button">
    <Button variant="contained">Contained</Button>
    <Button variant="outlined">Outlined</Button>
    <Button variant="text">Text</Button>
    <Button variant="contained" disabled>
      Contained
    </Button>
    <Button variant="outlined" disabled>
      Outlined
    </Button>
    <Button variant="text" disabled>
      Text
    </Button>
  </Story>
</Canvas>

## Radio

Radio buttons allow the user to select one option from a set.

```javascript
import { Radio } from '@terenceodonoghue/react-components/velocity';
```

Text can be placed to the right of the input using the `label` prop, alongside any standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes, e.g. `name`, `defaultChecked` or `disabled`.

<Canvas>
  <Story name="Radio">
    <Radio label="Shelob" name="radio-group-1" defaultChecked />
    <Radio label="Denethor" name="radio-group-1" />
    <Radio label="Quickbeam" name="radio-group-2" defaultChecked disabled />
    <Radio label="Shadowfax" name="radio-group-2" disabled />
  </Story>
</Canvas>

## Slider

Sliders allow users to make selections from a range of values.

```javascript
import { Slider } from '@terenceodonoghue/react-components/velocity';
```

Standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes are supported, e.g. `min`, `max`, `step` or `disabled`.

<Canvas>
  <Story name="Slider">
    <Slider />
    <Slider min={0} max={100} step={25} />
    <Slider disabled />
  </Story>
</Canvas>

## Switch

Switches toggle the state of a single setting on or off.

```javascript
import { Switch } from '@terenceodonoghue/react-components/velocity';
```

Standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes are supported, e.g. `defaultChecked` or `disabled`.

<Canvas>
  <Story name="Switch">
    <Switch />
    <Switch defaultChecked />
    <Switch defaultChecked disabled />
    <Switch disabled />
  </Story>
</Canvas>

## TextField

Text fields let users enter and edit text.

```javascript
import { TextField } from '@terenceodonoghue/react-components/velocity';
```

Text can be placed above the input using the `label` prop, alongside any standard [input](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input) attributes, e.g. `placeholder`, `type` or `disabled`.

<Canvas>
  <Story name="TextField">
    <TextField label="Label" />
    <TextField label="Email Address" placeholder="samplemail@mail.com" />
    <TextField label="Password" type="password" />
    <TextField label="Disabled" disabled />
  </Story>
</Canvas>

## Avatar

An avatar component represents an object or entity, typically a user's profile picture.

```javascript
import { Avatar } from '@terenceodonoghue/react-components/velocity';
```

The `variant` prop defines one of two styles: rounded (default) or square. Standard [img](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img) attributes are also supported, e.g. `alt`.

<Canvas>
  <Story name="Avatar">
    <Avatar alt="hello" src={faker.image.avatar()} />
    <Avatar src={faker.image.avatar()} variant="square" />
  </Story>
</Canvas>

Height and width can be customised by providing a number to the `size` prop.

<Canvas>
  <Story name="AvatarSize">
    <Avatar size={32} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={48} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={64} src={faker.image.avatar()} variant="rounded" />
    <Avatar size={80} src={faker.image.avatar()} variant="rounded" />
  </Story>
</Canvas>

## Pill

Pills are simple labels with a background color.

```javascript
import { Pill } from '@terenceodonoghue/react-components/velocity';
```

Set the text value using the `label` prop, alongside `color` to set both the text and backdrop colours. The backdrop has a default opacity of `0.2`, but this can be adjusted with the `opacity` prop.

<Canvas>
  <Story name="Pill">
    <Pill color="#2E5BFF" label={76} />
    <Pill color="#8C54FF" label={69} />
    <Pill color="#00C1D4" label={28} />
    <Pill color="#F7C137" label={198} />
  </Story>
</Canvas>

## Card

Cards contain content and actions about a single subject.

```javascript
import { Card } from '@terenceodonoghue/react-components/velocity';
```

An optional `caption` prop renders text above the content block.

<Canvas>
  <Story name="Card">
    <Card caption="Caption">Content</Card>
    <Card>Content</Card>
  </Story>
</Canvas>

The `color` prop sets the icon colour, or defaults to `currentColor` if left blank.

## Backdrop

A backdrop can be used with icons for added emphasis.

```javascript
import { Backdrop } from '@terenceodonoghue/react-components/velocity';
```

The `color` prop sets the backdrop colour with a default opacity of `0.15`. This can be adjusted with the `opacity` prop.
