# EyeDropper API

The [EyeDropper API](https://developer.mozilla.org/en-US/docs/Web/API/EyeDropper_API) provides a mechanism for creating an eyedropper tool. Using this tool, users can sample colors from their screens, including outside of the browser window.

## useColour

```javascript
import { useColour } from '@terenceodonoghue/react-hooks';
```

<Canvas>
