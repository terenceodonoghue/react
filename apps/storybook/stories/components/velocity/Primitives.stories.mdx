# Primitives

Primitives are the foundational visual elements of a design system that, when changed, can alter the perception of the brand. Components are comprised of design primitives and other, smaller components.

## Layout

### Pane

A pane is a simple wrapper for separating content.

```javascript
import { Pane } from '@terenceodonoghue/react-components/velocity';
```

The default padding is `20`, but this can be adjusted with `p`, `py` or `px`.

<Canvas>
  <Story name="Pane">
    <Pane>Beep</Pane>
    <Pane p={48}>Boop</Pane>
    <Pane py={48}>Beep</Pane>
    <Pane px={48}>Boop</Pane>
  </Story>
</Canvas>

## Text

Use typography to present your design and content as clearly and efficiently as possible.

```javascript
import { Text } from '@terenceodonoghue/react-components/velocity';
```

The `variant` prop sets the style. The available variants are: `h1`, `h2`,`h3`,`h4`,`p1` (default),`p2`,`p3`,`c1`,`c2`.

<Canvas>
  <Story name="Text">
    <Text variant="h1">Heading 1</Text>
    <Text variant="h2">Heading 2</Text>
    <Text variant="h3">Heading 3</Text>
    <Text variant="h4">Heading 4</Text>
    <Text variant="p1">Paragraph 1</Text>
    <Text variant="p2">Paragraph 2</Text>
    <Text variant="p3">Paragraph 3</Text>
    <Text variant="c1">Caption 1</Text>
    <Text variant="c2">Caption 2</Text>
  </Story>
</Canvas>

A `p` element is rendered by default, unless specified otherwise by `as`. This only affects the semantic element, and is independent of the style specified by `variant`. This allows elements and styles to be applied independently.

<Canvas>
  <Story name="TextAs">
    <Text as="h3">Heading h3 (as="h3")</Text>
    <Text as="h3" variant="h3">
      Heading h3 (as="h3" variant="h3")
    </Text>
  </Story>
</Canvas>

## Icons

Icons can be used to improve visual interest and draw attention to an element.

```javascript
import { Icon } from '@terenceodonoghue/react-components/velocity';
```

<IconGallery>
  {Object.entries(VelocityIcons).map(([name, Icon]) => (
    <IconItem name={name.toLowerCase()}>
      <Icon />
    </IconItem>
  ))}
</IconGallery>

Set the icon name using `with`, and provide an optional `color`. This defaults to `currentColor` if left blank.

<Canvas>
  <Story name="Icons">
    <Icon with="dashboard" />
    <Icon color="#2E5BFF" with="analytics" />
    <Icon color="#33AC2E" with="vehicles" />
    <Icon color="#8C54FF" with="service" />
    <Icon color="#D63649" with="map" />
    <Icon color="#00C1D4" with="mail" />
    <Icon color="#F7C137" with="settings" />
  </Story>
</Canvas>

A `backdrop` can be used for added emphasis.

<Canvas>
  <Story name="IconsBackdrop">
    <Icon backdrop color="#33AC2E" with="check" />
    <Icon backdrop color="#00C1D4" with="marker" />
    <Icon backdrop color="#8C54FF" with="energy" />
    <Icon backdrop color="#F7C137" with="dynamic" />
  </Story>
</Canvas>

Icon size is `24` by default (or `20` with a backdrop) but can be adjusted with the `size` prop.

<Canvas>
  <Story name="IconsSize">
    <Icon with="vehicles" />
    <Icon with="vehicles" size={32} />
    <Icon with="vehicles" size={40} />
    <Icon with="vehicles" size={48} />
  </Story>
</Canvas>

There's also a `compact` size preset that reduces the default icon size and (if present) its backdrop.

<Canvas>
  <Story name="IconsCompact">
    <Icon backdrop color="#33AC2E" size="compact" with="check" />
    <Icon backdrop color="#00C1D4" size="compact" with="marker" />
    <Icon backdrop color="#8C54FF" size="compact" with="energy" />
    <Icon backdrop color="#F7C137" size="compact" with="dynamic" />
  </Story>
</Canvas>
