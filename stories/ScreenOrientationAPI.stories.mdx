# Screen Orientation API

The [Screen Orientation API](https://developer.mozilla.org/en-US/docs/Web/API/Screen_Orientation_API) provides information about the orientation of the screen.

## useOrientation

```javascript
import { useOrientation } from '@terenceodonoghue/react-hooks';
```

<Canvas>
  <Story name="useOrientationHook">
    {() => {
      const { orientation, isLandscape, isPortrait } = useOrientation();
      return (
        <>
          <p>Current orientation: {orientation}</p>
          <p>Is landscape? {isLandscape.toString()}</p>
          <p>Is portrait? {isPortrait.toString()}</p>
        </>
      );
    }}
  </Story>
</Canvas>
